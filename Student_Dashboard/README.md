#React Student Scoreboard

This is a simple React project built using Vite. It displays student records and allows adding and updating their scores.

#Features

View student name and score
Add new students
Update scores
Display Pass/Fail status
Pass ≥ 40
Fail < 40

#Tech Stack

React (Vite)
JavaScript (JSX)
CSS
